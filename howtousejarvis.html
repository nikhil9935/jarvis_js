About Jarvis

- Developed voice assistant 'Jarvis' using HTML, CSS, and JavaScript.
- Integrated Web Speech API to enable vocal interaction.
- Implemented voice-controlled features, enabling task execution through natural voice commands.

How to Use Jarvis

To interact with Jarvis, use the following commands:

  • Jarvis / Hii Jarvis / Hello Jarvis / Hey Jarvis - Activate Jarvis: Activate Jarvis and start voice recognition.
  • show me commands: Display a list of available commands to use with Jarvis.
  • close this: Close the current tab.
  • change my information: Change information entered during the startup, such as GitHub profile, Instagram profile, etc.
  • whats the weather or temperature: Show the current weather details.
  • show the full weather report: Show the weather forecast.
  • Open chatroom:Chatroom is a place where you can chat with random peoples.
  • Open Quiz:You can give random questions quiz here.
  • Open News:You can read all category news here Ex. Entertainment,Sports,Trending News etc.
  • Open Joke:You can read random jokes here.
  • are you there: Check Jarvis's presence.
  • shut down: Stop voice recognition and deactivate Jarvis.
  • open google: Open Google website.
  • search for "your keywords": Search for specific keywords on Google.
  • open whatsapp: Open WhatsApp website or application.
  • open youtube: Open YouTube website.
  • play "your keywords": Search for specific keywords on YouTube and play relevant videos.
  • close this youtube tab: Close the opened YouTube tab.
  • open firebase: Open Firebase website.
  • open netlify: Open Netlify website.
  • open leetcode: Open Leetcode website.
  • open my leetcode profile: Open your leetcode profile.
  • open codechef: Open Codechef website.
  • open my codechef profile: Open your codechef profile.
  • open github: Open GitHub website.
  • open my github profile: Open your GitHub profile.
  • open facebook: Open Facebook.
